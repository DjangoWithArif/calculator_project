## Calculator Project

This is a simple Python CLI Calculator that allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. It is built using basic Python concepts like functions and loops.

### Features

- Addition
- Subtraction
- Multiplication
- Division
- Handles invalid inputs without crashing

### Run the Project

```bash
python calculator_app.py
```


#### NOTE: To use the OOP Calculator, run:

```bash
python cal_oop.py
```
